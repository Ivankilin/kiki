# IOTA Messenger

Это мессенджер, использующий IOTA для обмена сообщениями.

## Установка

1. Клонируйте репозиторий
2. Выполните `npm install` для установки всех зависимостей

## Запуск

Для запуска проекта выполните `npm start`
